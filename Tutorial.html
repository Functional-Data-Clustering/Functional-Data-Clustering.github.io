REM Tabular Here coming out soon The Reinforced EM (REM) algorithm provides an efficient solution to the initilization problem of the EM algorithm for clustering with Gaussian mixture models.

DCF Tabular Here GitHub Density Core Finding (DCF) is able to detect clusters of varying density and irregular shape, and applicable to big data with numerous clusters.
The idea is to detect high-density core regions, each region representing a cluster, and then assign each non-core point to the same cluster as its nearest neighbor of higher density.
